Notion

- strong 태그 => 굵은 글씨 (강조)

- h1~h6 태그 => 제목? 그런 느낌 굵은 글씨, 줄바꿈이 자동

- u 태그 => underline 밑줄 - 별로 안중요함 잊어도 됨

- br 태그 => 줄바꿈 태그

- p 태그 => paragraph 태그, 문단을 나누는데에 사용

br, p 둘 다 줄바꿈에 사용하지만, p가 정보로서의 가치를 의미론적으로 더 잘 표현 할 수 있고 css를 통해 단순 줄바꿈이 아닌 더 정교한 조정도 가능하기 때문에 p를 쓰는 것이 더 좋다고 볼 수 있음

어떤 단어를 css 또는 이미지로 만져서 웹사이트 제목으로 쓰는 경우와 h 태그를 사용해서 제목으로 쓰는 경우를 비교하면 h 태그를 사용했을 때 검색 엔진에 잘 잡힘

- img 태그는 말 그대로 웹페이지에 이미지를 띄우는 태그, 하지만 태그명만으로는 무슨 이미지인지, 크기가 어떤지 알 수 없으므로 src(source)나 width 같은 속성을 이용해서 사용함

- ul은 unordered list, ol은 ordered list li는 list 태그임
ul ol과 li는 부모자식 관계를 갖는 태그로 둘은 항상 함께 와야함

*css를 배우면 노션처럼 깔끔하게 만들 수 있을텐데 일단은 이렇게라도 복습겸 실습으로 만들어봅세다
